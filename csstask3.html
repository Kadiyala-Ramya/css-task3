Tim Berners-Lee

Tim Berners-Lee (1955– )

— Inventor of the World Wide Web

Tim Berners-Lee is a British computer scientist best known for inventing the World Wide Web in 1989. His revolutionary idea was to create a system that allowed researchers to access and share documents over the internet using hyperlinks.

His Inventions

HTML (HyperText Markup Language) – the standard language for creating web pages.

HTTP (HyperText Transfer Protocol) – the communication protocol for transferring web pages.

URL (Uniform Resource Locator) – the address system for web resources.
